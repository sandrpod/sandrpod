AI 编码助手的挑战

代码执行是 AI 编程助手的核心能力，

但现有方案面临诸多挑战：

• 安全问题 — 在宿主机直接执行用户代码，风险极高

• 环境隔离 — 不同任务需要独立环境，状态不能互相污染

• 状态保持 — AI 需要在执行环境中有持久化的工作目录

• 快速启动 — 每个请求都创建新容器太慢

什么是 SandrPod？

SandrPod 是一个专为 AI Agent 设计的代码执行沙箱平台，

基于容器化技术，提供安全隔离、状态持久化、快速响应的代码执行环境。

更轻量、更简化的沙箱架构，

无缝集成 LangChain DeepAgents 等主流 AI Agent 框架。

应用场景

🤖 AI 编程助手

为 AI Agent 提供安全的代码执行环境，实现代码编写、调试、测试的完整闭环。

🔬 代码分析与审查

在隔离环境中执行代码分析工具，检查代码质量、安全漏洞。

📦 自动化测试

为每个测试用例创建独立沙箱环境，并行执行，互不干扰。

🌐 远程开发环境

提供云端开发环境，支持多用户、多项目、持久化存储。

快速开始

$ pip install sandrpod-cli
安装 CLI 工具

$ sandrpod-cli session create
创建持久化 Session

$ sandrpod-cli session exec "cd /project && python main.py"
在 Session 中执行命令（状态保持）

$ sandrpod-cli fs ls /project
文件操作
开始使用 SandrPod
